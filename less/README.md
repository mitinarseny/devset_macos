# less

> [`less`](http://man7.org/linux/man-pages/man1/less.1.html) is a standart unix pager

## Key bindings

### Navigation

| Key  | Description  |
|:----:|--------------|
| `g`  | go to top    |
| `G`  | go to bottom |


### Search

| Key              | Description          |
|:----------------:|----------------------|
| `/ + <pattern>`  | search pattern       |
| `n`              | next search item     |
| `N`              | previous search item |
