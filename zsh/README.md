# ZSH

> [ZSH](http://zsh.org) is POSIX-compatible shell designed for interactive use.

## Features

* [aliases](aliases.zsh)
* [completion](completion.zsh)
* [functions](functions)
  * [`mkc some/dir`](functions/mkc): mkdir and cd
  * [`backup some/file`](functions/backup): move to `some/file~`

## Install

```sh
make
```
